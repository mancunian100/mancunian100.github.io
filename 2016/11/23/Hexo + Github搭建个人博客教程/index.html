    Hexo + Github搭建个人博客教程

    2016-11-23

    前言

    想搭建一个个人博客，但是不舍得花钱买服务器，听说github可以有免费的搭建博客的方法，看到有各种教程关于hexo、jelly和wordpress框架的，我自己选择了hexo，一路走下来感觉有很多坑（当然是对我这样一个连github都还不熟悉的新手来说），于是把走过的坑整理了一下，方便以后再来一次，也方便后来的人

    所以，这样搭建博客好处是什么”)所以，这样搭建博客好处是什么

    关于github：

    • github pages是用来托管 github 上静态网页的免费站点，有足够的免费空间（1G），资料自己管理，保存可靠；
    • 享受github 的便利，上面有很多大牛，眼界会开阔很多；
    • 可以使用很多现成的博客框架，搭建过程简单快速；

    关于hexo：

    • Hexo 是一个简单、快速、强大的静态博客框架,出自台湾大学生 tommy351 之手；
    • 极速生成静态页面；
    • 一键部署博客；
    • 丰富的插件支持；
    • 支持 Markdown
