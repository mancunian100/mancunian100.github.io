    django配置mysql版本问题报错

    2019-06-13

    在django服务开启之后需要跑通与mysql数据库的连接，这里遇到了版本问题

    环境：python3.6+django2.2

    • 最开始检查环境时用pip升级了下django到最新的2.2，却发现由此引来了版本问题

    • 在配置数据库时会报错

    • 方案一：一开始查到的解决方案，由于django在2.2下会检查mysqlclient的版本，如果注释掉其中的版本检查就可以了

    • 于是就可以正常配置mysql

    • 方案二：后来知道我们的开发环境都是用的django2.1.4，只要把django版本降回来就自然没有这个版本的报错了
